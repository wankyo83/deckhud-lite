# DeckHUD Lite

Steam Deck·SteamOS용 Decky Loader HUD 플러그인입니다. Steam의 MangoApp HUD를 이용해 필요한 정보만 간단하게 표시합니다.

## 기능

- 배터리 잔량(%), 현재 소비전력(W), 예상 남은 시간
- 현재 시각과 선택적 FPS 표시
- 팬 속도, 화면 주사율, 설정된 FPS 제한 표시
- 수평·수직 HUD와 표시 항목 순서 변경
- 위치, 글자 크기, 배경 진하기 조절
- 원래 Steam HUD 설정 백업 및 복원

## 설치

1. [최신 Release](https://github.com/wankyo83/deckhud-lite/releases/latest)에서 `deckhud-lite.zip`을 받습니다.
2. Decky Loader 설정에서 개발자 모드를 켭니다.
3. `Install Plugin from ZIP File`을 선택하고 받은 ZIP을 설치합니다.

게임에서 Steam 성능 오버레이를 켠 뒤 `…` → Decky → DeckHUD Lite에서 설정할 수 있습니다.
